
# 黑底閱讀器 V3K FIX（完整包：全域套用 + 防快取）

**這包完全沿用你的原始檔案結構**，只改：
- `script.js`：設定頁切換會 **寫入 localStorage**，回到閱讀器也跟著變。
- `settings.html`：新增/更新為外觀設定頁。
- `sw.js`：更換快取名稱 `reader-v3k-full-v1758358748`，確保新檔生效。

## 安裝／還原
1. 先備份你的站台資料夾。
2. 解壓這包 → **整包覆蓋**到站台目錄。
3. 打開 `settings.html` 切換一次模式（會儲存）。
4. 回 `index.html`，重新整理一次；若是 PWA，請關掉再開（讓 SW 更新）。

## 如果你之後要「只刷新快取」
打開 `sw.js`，把第一行的 `CACHE` 名稱改個新值即可（例如 `-v3`）。
